# 출석 관리 시스템 - Python GUI 버전

PyQt5 기반 데스크톱 애플리케이션

## 📋 기능

- ✅ 시작일 자동 설정 (1개월 단위)
- ✅ 캘린더 기반 직관적인 UI
- ✅ 6가지 출결 상태 (출석/결석/지각/조퇴/외출/병결)
- ✅ 실시간 출석률 계산
- ✅ 복잡한 패널티 규칙 적용
  - 동일 사유 2회 = 결석 1회
  - 상이 사유 3회 = 결석 1회
- ✅ 목표 출석률 설정 (100%, 95%, 90%, 85%, 80%, 75%)
- ✅ 통계 대시보드
- ✅ EXE 파일로 배포 가능

## 🚀 설치 및 실행

### 방법 1: Python 스크립트로 실행

1. **필요한 패키지 설치:**
   ```bash
   pip install -r requirements.txt
   ```

2. **앱 실행:**
   ```bash
   python attendance_app.py
   ```

### 방법 2: EXE 파일로 빌드

1. **EXE 빌드:**
   ```bash
   # Windows
   build_exe.bat
   ```

2. **생성된 EXE 실행:**
   ```
   dist\출석관리시스템.exe
   ```

## 💡 사용 방법

1. **앱 실행**
   - 자동으로 오늘부터 1개월 단위 기간 설정됨

2. **출결 상태 변경**
   - 캘린더에서 평일 날짜 클릭
   - 팝업 창에서 상태 선택 (출석/결석/지각/조퇴/외출/병결)
   - 확인 버튼 클릭

3. **출석률 확인**
   - 상단에 큰 숫자로 현재 출석률 표시
   - 통계 카드에서 각 상태별 횟수 확인

4. **목표 출석률 설정**
   - 우측 상단 드롭다운에서 목표율 선택
   - 달성 여부에 따라 색상 변경

## 📊 출석률 계산 규칙

### 기본 규칙
- **총 수업일수**: 평일(월~금)만 카운트
- **병결**: 출석으로 인정 (감점 없음)

### 패널티 규칙

**1. 동일 사유 중복 (Same Type Pair)**
- 지각 2회 = 결석 1회
- 조퇴 2회 = 결석 1회
- 외출 2회 = 결석 1회

**2. 상이 사유 중복 (Mixed Type Triple)**
- 규칙 1 적용 후 남은 횟수의 합
- (지각 나머지 + 조퇴 나머지 + 외출 나머지) 3회 = 결석 1회

### 계산 공식
```
최종 결석 수 = 직접 결석 + (규칙1 환산 결석) + (규칙2 환산 결석)
출석률(%) = ((총 수업일수 - 최종 결석 수) / 총 수업일수) × 100
```

## 🎨 UI 특징

- **모던한 디자인**: 깔끔한 카드 기반 레이아웃
- **색상 코딩**: 각 상태별 고유 색상
  - 출석: 초록색 (#10b981)
  - 결석: 빨간색 (#ef4444)
  - 지각: 주황색 (#f97316)
  - 조퇴: 노란색 (#eab308)
  - 외출: 연보라색 (#a78bfa)
  - 병결: 회색 (#6b7280)
- **캘린더 하이라이트**: 날짜별 상태가 색상으로 표시
- **실시간 업데이트**: 상태 변경 시 즉시 반영
